Media Queries

¿Qué son los Media Queries?

Los Media Queries son una característica de CSS3 que permite aplicar estilos específicos basados en las características del dispositivo, como el ancho de la pantalla, la orientación o la resolución.

¿Cómo se usan?

Los Media Queries se utilizan dentro de las hojas de estilo CSS o en la sección <style> de un documento HTML. La sintaxis básica es: 

@media tipo-de-medio and (característica: valor) {
    /* Estilos CSS */
}

¿Dónde se usan?

Los Media Queries se usan principalmente en:

  • Diseño responsive para adaptar sitios web a diferentes tamaños de pantalla
  • Optimización de presentación para diferentes dispositivos (móviles, tablets, laptops)
  • Ajuste de estilos basados en las capacidades del dispositivo como orientación o resolución
  • Creación de versiones para impresión de páginas web
